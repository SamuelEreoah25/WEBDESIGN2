MY COURSES DESCRIPTON LOCATION
Problem Solving and Programming This part of the module introduces students to the skills, principles and concepts necessary to solve problems in computing.
It also aims to develop essential skills to enable the solution of these problems with the construction of appropriate algorithms and a computer program. Principles underlying the design of a high level programming language will be introduced.
DATABASE Databases 1 is a hands-on module that applies data modelling techniques to establish, modify and maintain data structures and associated components such as entities, relationships and attribute definitions. Students undertake design of a databases applications and translate and implement development requirements into physical database structures using industry desirable software and languages such as Oracle, MySQL and Structured Query Language (SQL).
The module highlights the application of industry standards and good practice to optimise employability and performance.		 Hotbeans office, Maddocks, Cardiff
